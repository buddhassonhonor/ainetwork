# AI-Network: 智能化计算机网络教学平台 🚀

基于人工智能技术深度赋能的计算机网络课程教学平台，由 **泉州师范学院物理与信息工程学院计算机网络核心教研组** 开发。

## 🌟 核心亮点

### 1. 班级学业全景星系图 🌌
* **动态引力图谱**：采用 `react-force-graph-2d` 渲染，将学生学情数据可视化为动态漂浮的星球。
* **多维数据画像**：星球大小由学习进度决定，颜色代表综合评分，支持实时交互与悬浮诊断预览。
* **智能风云榜**：实时同步班级尖子生动态，激励学习竞争。

### 2. AI+ 知识图谱 🧠
* **五层协议栈融合**：深度整合传统计网五层协议栈与 AI 赋能节点。
* **交互式探索**：支持动态缩放与居中适配，点击节点即可获取 AI 在该协议层的应用案例与学习资源。
* **高占比 AI 模块**：AI 节点占比达 45%，构建智能驱动的现代化教学体系。

### 3. 个性化学生诊断看板 📊
* **AI 诊断结论**：基于学习轨迹自动生成诊断建议，识别知识薄弱点。
* **多维能力雷达图**：直观展示学生在物理层、网络层、传输层等各维度的能力分布。
* **活跃度追踪**：周度学习热度趋势分析，助力精准导学。

## 🛠️ 技术栈

* **前端框架**: React 19 + Vite
* **样式处理**: Tailwind CSS (v4)
* **动效交互**: Framer Motion
* **数据可视化**: Recharts + React Force Graph
* **图标库**: Lucide React
* **部署平台**: Vercel

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📂 项目结构
* `src/pages/Dashboard.jsx`: 班级全景与个人学情看板
* `src/pages/KnowledgeGraph.jsx`: AI+ 知识图谱交互页面
* `src/components/Navbar.jsx`: 具备滚动自动隐藏与背景切换功能的智能导航栏
* `src/data/students.js`: 模拟学情数据中心
